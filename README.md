# 🎬 Movie Ratings Analysis Dashboard

📌 Project Overview

This project focuses on analyzing movie data using Python and Tableau.
The main goal was to perform data cleaning, exploratory data analysis (EDA), and create interactive visualizations & dashboards.

🚀 Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Tableau
* Jupyter Notebook

📊 Features
* IMDB Rating Distribution
* Top 10 Rated Movies
* Genre Analysis
* Movies Released Per Year
* Interactive Dashboard

📈 Visualizations
The project includes:
* Histogram
* Bar Charts
* Area Chart
* Dashboard Visualizations

🎯 Learning Outcome
Through this project, I improved my skills in:
* Data Cleaning
* Data Visualization
* Dashboard Designing
* Tableau & Python Integration
